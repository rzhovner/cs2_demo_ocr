# Experiments

Hypothesis-driven experiment reports. Each report follows this structure:

1. **Hypothesis** -- A testable prediction
2. **Method** -- What data was collected and how it was analyzed
3. **Results** -- Metric-by-metric findings
4. **Verdict** -- Supported / refuted / inconclusive, with explanation
